# chuna-ai

TypeScript npm package skeleton.

## Install

```bash
npm i chuna-ai
```

## Usage

```ts
import { greet } from 'chuna-ai';

console.log(greet('World'));
// => Hello, World!
```

## Scripts

- build: Bundles with tsup to dist/
- dev: Watch mode
- typecheck: Run TypeScript
- lint: ESLint
- test: Vitest
